## 这是什么

一本**实验驱动**的具身智能入门课。每个实验是一个独立的 Python 项目——可运行、有可视化、每一步你都亲手写出来。

大多数具身智能学习资源告诉你"领域里有什么、去哪学"。这门课做的是另一件事：**带你亲手做出来。**

> 你写的不是笔记，是代码。你得到的不是"我听说过"，是"我的机器人真的跑起来了。"

## 实验怎么串起来

```
01(KF) ●
  │
  ├──→ 04(仿真) ──→ 05(PID) ──→ 07(规划) ──→ 08(轨迹优化)
  │       │            │              │
  │       │            │              └──→ S1(全栈导航)
  │       │            │
  │       ├──→ 09(QLearn) ──→ 10(DRL) ──→ 11(模仿学习)
  │       │
  │       ├──→ 06(运动学) ──→ 12(VLA)
  │       │
  │       └──→ 12(VLA)
  │
  └──→ 03(EKF-SLAM)
```

**04（仿真世界）是关键节点** — 几乎所有后续实验都跑在它上面。它建好了，后面的实验构建成本减半。

## 每个实验长什么样

```
experiments/01-kalman-filter/
├── tutorial.md          问题 -> 直觉 -> 代码 -> 公式 -> 实验
├── requirements.txt     numpy + matplotlib（尽量零额外依赖）
├── filters/             核心算法，可直接 import
├── demos/               一键运行的 demo 脚本
├── utils/               可视化 + 仿真数据生成
└── output/              生成的图片和动画
```

**教学循环**：动手构建 → 观察行为 → 理解原理 → 调参实验 → 形成直觉。

## 建设计划

| 批次 | 实验 | 为什么先做它们 |
|:--:|------|--------------|
| 1 | 04 仿真 + 05 PID | 04 是共享基础设施，05 补齐"行动"，和 KF 构成最小闭环 |
| 2 | 02 MCL + 07 规划 | 复用 KF 代码，低成本高价值，可视化效果极强 |
| 3 | 03 EKF-SLAM + 09 Q-Learning | 打破 SLAM 恐惧 + 吸引学习者 |
| 4 | 06 运动学 + 08 轨迹 + 10 DRL | 解锁机械臂和深度强化学习 |
| 5 | 11 模仿学习 + 12 VLA | 前沿话题，GPU 可选 |
| 6 | S1 全栈导航 | 收尾大项目，串联一切 |
