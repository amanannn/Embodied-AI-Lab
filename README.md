## 📌 简介

本项目记录我在具身智能领域的学习与实践，包括：
- 在仿真环境（如 AI2-THOR、Habitat、Gibson）中训练智能体
- 实现导航、目标定位、交互任务等基础能力
- 探索强化学习（RL）、模仿学习（IL）与世界模型在具身场景中的应用

目标：构建一个可扩展、模块化的具身智能实验框架。
