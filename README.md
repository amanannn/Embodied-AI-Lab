## 这是什么

一本**实验驱动**的具身智能入门课。每个实验是一个独立的项目——可运行、有可视化、每一步你都亲手写出来。

大多数具身智能学习资源告诉你"领域里有什么、去哪学"。这门课做的是另一件事：**带你亲手做出来。**

> 你写的不是笔记，是代码。你得到的不是"我听说过"，是"我的机器人真的跑起来了。"

## 每个实验长什么样

```
experiments/XX-name/
├── README.md             问题 -> 直觉 -> 代码 -> 公式 -> 实验
├── requirements.txt      尽量零额外依赖（Python）/ CMakeLists.txt（C++）
├── src/ 或 lib/          核心实现，可直接 import / #include
├── demos/ 或 app/        一键运行的 demo
├── utils/                可视化 + 辅助工具
└── output/               生成的图片和动画
```

**教学循环**：动手构建 → 观察行为 → 理解原理 → 调参实验 → 形成直觉。
