# legacy-experiments — Legacy Experiment Archive

中文： [README.md](./README.md)

This directory stores legacy experiments from before the direction-first restructure. They are kept to:

- serve as source material for future migration into the new `experiments/` direction structure
- preserve historical experiment content for search and traceability
- avoid mixing legacy experiment numbers with new direction numbers under top-level `experiments/`

For current learning, start from `experiments/`. See the migration mapping here:

- [Legacy-to-Direction Mapping](../../docs/curriculum/legacy-to-direction-map.en.md)
