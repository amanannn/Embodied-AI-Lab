# archive — Historical Content

中文： [README.md](./README.md)

`archive/` stores historical content that remains useful for reference but is no longer the current learning entry point.

Current public course entry points are:

- `experiments/`: the 10 direction-first curriculum entries
- `shared/`: reusable cross-direction infrastructure
- `docs/curriculum/`: course path, migration mapping, and structure documentation

Legacy experiments are not deleted. They are moved to `archive/legacy-experiments/` as migration sources, historical references, and future refactoring material.
