# RL 与模仿学习 (RL and Imitation) — Level 3: Research Extension

English: [README.en.md](./README.en.md)

## 定位

本目录是强化学习与模仿学习方向的 **研究扩展层**。Level 3 连接课程版本与研究问题、论文和进一步实现路径，不承诺空泛的"完整实现"，而是明确研究边界。

## 研究方向

| 方向 | 研究问题 | 前置要求 |
|------|---------|---------|
| Sim-to-Real RL | 如何将仿真中训练的策略迁移到真实机器人？ | Level 1 + Level 2 策略部署 |
| 离线 RL | 如何从离线数据集中学习策略而不交互？ | Level 1 Q-learning / 深度 RL |
| 安全探索 | 如何在学习过程中保证机器人安全？ | Level 2 实时控制接口 |

## 与 Level 1/2 的关系

- Level 1 建立策略学习的算法直觉
- Level 2 将策略算法接入 ROS2 / C++ 真实系统
- Level 3 探索更开放的研究问题和扩展边界
