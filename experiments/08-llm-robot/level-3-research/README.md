# 大模型与机器人 (LLM and Robotics) — Level 3: Research Extension

English: [README.en.md](./README.en.md)

## 定位

本目录是大模型与机器人方向的 **研究扩展层**。Level 3 连接课程版本与研究问题、论文和进一步实现路径，不承诺空泛的"完整实现"，而是明确研究边界。

## 研究方向

| 方向 | 研究问题 | 前置要求 |
|------|---------|---------|
| 扎根规划 | 如何让 LLM 的规划扎根于可执行的机器人行为？ | Level 1 任务分解 + Level 2 执行接口 |
| 多模态智能体 | 如何构建整合视觉、语言和动作的智能体？ | Level 1 + 深度学习基础 |
| 长期任务编排 | 如何编排跨越长时间尺度的机器人任务？ | Level 1 + Level 2 系统集成 |

## 与 Level 1/2 的关系

- Level 1 建立 LLM 规划的算法直觉
- Level 2 将规划算法接入 ROS2 / C++ 真实系统
- Level 3 探索更开放的研究问题和扩展边界
