# LLM and Robotics — Level 3: Research Extension

中文： [README.md](./README.md)

## Positioning

This directory is the **research extension layer** for the LLM and robotics direction. Level 3 connects course versions to research questions, papers, and deeper implementation paths — it does not promise vague "complete implementations" but clarifies research boundaries.

## Research Directions

| Direction | Research Question | Prerequisites |
|-----------|------------------|---------------|
| Grounded Planning | How to ground LLM planning in executable robot behavior? | Level 1 task decomposition + Level 2 execution interfaces |
| Multimodal Agents | How to build agents that integrate vision, language, and action? | Level 1 + deep learning foundations |
| Long-horizon Task Orchestration | How to orchestrate robot tasks across long time scales? | Level 1 + Level 2 system integration |

## Relationship to Level 1/2

- Level 1 builds LLM planning algorithm intuition
- Level 2 connects planning algorithms into ROS2 / C++ real systems
- Level 3 explores more open research questions and extension boundaries
