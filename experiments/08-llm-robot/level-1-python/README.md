# 大模型与机器人 (LLM and Robotics) — Level 1: Core Python Lab

English: [README.en.md](./README.en.md)

## 定位

本目录是大模型与机器人方向的 **Level 1 入口**。Level 1 是当前主产品：纯 Python 实现，无 ROS2 / Gazebo / Isaac / GPU 依赖，可在 Python 3.10+ 运行环境中直接运行。

## 规划中的实验

| 实验 | 核心问题 | 状态 |
|------|---------|------|
| 任务分解 | 如何用 LLM 将复杂任务拆解为可执行步骤？ | 规划中 |
| 工具调用 | 如何让 LLM 调用外部工具完成子任务？ | 规划中 |

## 与 Level 2 的关系

Level 1 用 Python 建立 LLM 规划直觉。Level 2（`../level-2-ros2-bridge/`）将这些算法接入 ROS2 / C++ / 真实机器人软件栈。
