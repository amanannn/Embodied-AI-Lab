## Positioning

This directory is the **Level 1 entry** for the vertical applications direction. Level 1 is the current main product: pure Python implementations with no ROS2 / Gazebo / Isaac / GPU dependencies, runnable in a Python 3.10+ runtime environment.

## Relationship to Level 2

Level 1 builds scenario-driven application intuition with Python. Level 2 (`../level-2-ros2-bridge/`) connects these exercises into ROS2 / C++ / real robot software stacks as integrated capstone projects.
