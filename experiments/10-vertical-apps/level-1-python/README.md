# 垂直行业应用 (Vertical Applications) — Level 1: Core Python Lab

English: [README.en.md](./README.en.md)

## 定位

本目录是垂直行业应用方向的 **Level 1 入口**。Level 1 是当前主产品：纯 Python 实现，无 ROS2 / Gazebo / Isaac / GPU 依赖，可在 Python 3.10+ 运行环境中直接运行。

## 规划中的实验

| 实验 | 核心问题 | 状态 |
|------|---------|------|
| 自主巡检 | 如何将感知、导航、控制整合为场景化系统？ | 规划中 |

## 与 Level 2 的关系

Level 1 用 Python 建立场景化应用直觉。Level 2（`../level-2-ros2-bridge/`）将这些练习接入 ROS2 / C++ / 真实机器人软件栈，形成集成式顶石项目。
