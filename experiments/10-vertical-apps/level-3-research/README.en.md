# Vertical Applications — Level 3: Research Extension

中文： [README.md](./README.md)

## Positioning

This directory is the **research extension layer** for the vertical applications direction. Level 3 connects course versions to research questions, papers, and deeper implementation paths — it does not promise vague "complete implementations" but clarifies research boundaries.

## Research Directions

| Direction | Research Question | Prerequisites |
|-----------|------------------|---------------|
| Domain-specific System Design | How to design embodied systems for specific industry constraints? | Level 1 + Level 2 system integration |
| Runtime Monitoring | How to monitor and maintain robot systems after deployment? | Level 2 operations interfaces |
| Human-Robot Collaboration | How to design safe and efficient human-robot collaborative systems? | Level 1 + safety constraint foundations |

## Relationship to Level 1/2

- Level 1 builds scenario-driven application intuition
- Level 2 connects applications into ROS2 / C++ real systems
- Level 3 explores more open research questions and extension boundaries
