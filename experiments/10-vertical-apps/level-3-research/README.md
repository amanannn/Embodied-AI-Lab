# 垂直行业应用 (Vertical Applications) — Level 3: Research Extension

English: [README.en.md](./README.en.md)

## 定位

本目录是垂直行业应用方向的 **研究扩展层**。Level 3 连接课程版本与研究问题、论文和进一步实现路径，不承诺空泛的"完整实现"，而是明确研究边界。

## 研究方向

| 方向 | 研究问题 | 前置要求 |
|------|---------|---------|
| 领域特定系统设计 | 如何针对特定行业约束设计具身系统？ | Level 1 + Level 2 系统集成 |
| 运行时监控 | 如何在部署后监控和维护机器人系统？ | Level 2 运维接口 |
| 人机协作 | 如何设计安全高效的人机协作系统？ | Level 1 + 安全约束基础 |

## 与 Level 1/2 的关系

- Level 1 建立场景化应用的直觉
- Level 2 将应用接入 ROS2 / C++ 真实系统
- Level 3 探索更开放的研究问题和扩展边界
