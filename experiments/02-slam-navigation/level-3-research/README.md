# SLAM 与导航 (SLAM & Navigation) — Level 3: Research Extension

English: [README.en.md](./README.en.md)

## 定位

本目录是 SLAM 与导航方向的 **研究扩展层**。Level 3 连接课程版本与研究问题、论文和进一步实现路径，不承诺空泛的"完整实现"，而是明确研究边界。

## 研究方向

| 方向 | 研究问题 | 前置要求 |
|------|---------|---------|
| 语义 SLAM | 如何将语义信息融入 SLAM 的地图表示？ | Level 1 + Level 2 SLAM 基础 |
| 动态场景导航 | 如何在有动态障碍物的环境中安全导航？ | Level 1 路径规划 + Level 2 导航集成 |
| 大规模地图构建 | 如何构建和维护大规模环境地图？ | Level 2 SLAM 前端 |

## 与 Level 1/2 的关系

- Level 1 建立路径规划和定位的算法直觉
- Level 2 将这些算法接入 ROS2 / C++ 真实系统
- Level 3 探索更开放的研究问题和扩展边界
