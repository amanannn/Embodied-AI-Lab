# 视觉语言导航 (Vision-Language Navigation) — Level 1: Core Python Lab

English: [README.en.md](./README.en.md)

## 定位

本目录是视觉语言导航方向的 **Level 1 入口**。Level 1 是当前主产品：纯 Python 实现，无 ROS2 / Gazebo / Isaac / GPU 依赖，可在 Python 3.10+ 运行环境中直接运行。

## 规划中的实验

| 实验 | 核心问题 | 状态 |
|------|---------|------|
| 语言目标导航 | 如何理解语言指令并找到语义目标？ | 规划中 |
| 语义搜索 | 如何在环境中搜索语义相关的物体？ | 规划中 |

## 与 Level 2 的关系

Level 1 用 Python 建立语言导航直觉。Level 2（`../level-2-ros2-bridge/`）将这些算法接入 ROS2 / C++ / 真实机器人软件栈。
