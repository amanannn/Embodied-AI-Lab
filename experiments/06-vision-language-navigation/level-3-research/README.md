# 视觉语言导航 (Vision-Language Navigation) — Level 3: Research Extension

English: [README.en.md](./README.en.md)

## 定位

本目录是视觉语言导航方向的 **研究扩展层**。Level 3 连接课程版本与研究问题、论文和进一步实现路径，不承诺空泛的"完整实现"，而是明确研究边界。

## 研究方向

| 方向 | 研究问题 | 前置要求 |
|------|---------|---------|
| 长期指令跟随 | 如何执行多步骤的长期语言指令？ | Level 1 语言导航 + 记忆机制 |
| 语义地图 | 如何构建支持语言查询的语义地图？ | Level 1 + SLAM 基础 |
| 多模态记忆 | 如何整合视觉、语言和空间记忆？ | Level 1 + 深度学习基础 |

## 与 Level 1/2 的关系

- Level 1 建立语言导航的算法直觉
- Level 2 将导航算法接入 ROS2 / C++ 真实系统
- Level 3 探索更开放的研究问题和扩展边界
