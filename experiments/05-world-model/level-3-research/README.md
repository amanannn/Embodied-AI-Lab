# 世界模型 (World Models) — Level 3: Research Extension

English: [README.en.md](./README.en.md)

## 定位

本目录是世界模型方向的 **研究扩展层**。Level 3 连接课程版本与研究问题、论文和进一步实现路径，不承诺空泛的"完整实现"，而是明确研究边界。

## 研究方向

| 方向 | 研究问题 | 前置要求 |
|------|---------|---------|
| 潜在世界模型 | 如何在低维潜在空间中学习世界动力学？ | Level 1 动力学预测 + 深度学习基础 |
| 基于模型的 RL | 如何用学到的世界模型加速策略学习？ | Level 1 + RL 基础 |
| 多模态预测 | 如何整合视觉、触觉等多模态信息进行预测？ | Level 1 + 感知基础 |

## 与 Level 1/2 的关系

- Level 1 建立动力学预测的算法直觉
- Level 2 将预测模型接入 ROS2 / C++ 规划系统
- Level 3 探索更开放的研究问题和扩展边界
