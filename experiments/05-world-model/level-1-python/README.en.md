# World Models — Level 1: Core Python Lab

中文： [README.md](./README.md)

## Positioning

This directory is the **Level 1 entry** for the world models direction. Level 1 is the current main product: pure Python implementations with no ROS2 / Gazebo / Isaac / GPU dependencies, runnable in a Python 3.10+ runtime environment.

## Planned Experiments

| Experiment | Core Question | Status |
|-----------|---------------|--------|
| Dynamics Prediction | How to learn forward models of the physical world? | Planned |
| Short-horizon Rollout | How to use prediction models to simulate future states? | Planned |

## Relationship to Level 2

Level 1 builds predictive modeling intuition with Python. Level 2 (`../level-2-ros2-bridge/`) connects these algorithms into ROS2 / C++ / real robot software stacks.
