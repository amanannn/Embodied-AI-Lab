## II. What is Image Super-Resolution

### 2.1 Problem Definition

The goal of image super-resolution: **reconstruct a high-resolution image from a low-resolution input**.

$$I_{HR} \approx f_\theta(I_{LR})$$

Where $I_{LR}$ is the low-resolution input, $f_\theta$ is the neural network, and $I_{HR}$ is the high-resolution target.

### 2.2 Degradation Model

Low-resolution images are typically produced from HR images through a degradation process:

$$I_{LR} = D(I_{HR}) = (I_{HR} * k) \downarrow_s + n$$

| Factor | Symbol | Description |
|--------|--------|-------------|
| Blur kernel | $k$ | Blur caused by optics or motion |
| Downsampling | $\downarrow_s$ | Reduce resolution by factor $s$ |
| Noise | $n$ | Sensor noise |

In this experiment, we simplify to **bicubic downsampling + bicubic upsampling** to construct LR-HR training pairs.

### 2.3 Evaluation Metrics

| Metric | Full Name | Meaning | Range |
|--------|-----------|---------|-------|
| PSNR | Peak Signal-to-Noise Ratio | Pixel-level reconstruction quality | Higher is better, typically 20-40 dB |
| SSIM | Structural Similarity | Structural similarity | 0~1, closer to 1 is better |

$$PSNR = 10 \cdot \log_{10}\left(\frac{MAX_I^2}{MSE}\right)$$

## III. Method Principles

### 3.1 Interpolation (Traditional Baseline)

```python
# Bicubic interpolation — traditional method, used as baseline
lr_image = cv2.imread('lr.png')
sr_bicubic = cv2.resize(lr_image, None, fx=2, fy=2, interpolation=cv2.INTER_CUBIC)
```

| Method | Pros | Cons |
|--------|------|------|
| Nearest Neighbor | Fast | Severe aliasing |
| Bilinear | Smooth | Blurry edges |
| Bicubic | Better smoothness | Loses high-frequency details |

### 3.2 CNN-based Super-Resolution

Convolutional neural networks can learn the LR→HR mapping from data, recovering high-frequency details that interpolation cannot:

```python
class SRCNN(nn.Module):
    """Classic Super-Resolution CNN"""
    def __init__(self):
        super().__init__()
        # Feature extraction
        self.conv1 = nn.Conv2d(3, 64, kernel_size=5, padding=2)
        # Non-linear mapping
        self.conv2 = nn.Conv2d(64, 64, kernel_size=3, padding=1)
        # Reconstruction
        self.conv3 = nn.Conv2d(64, 3, kernel_size=5, padding=2)
        self.relu = nn.ReLU()

    def forward(self, x):
        # Residual learning: learn HR - LR difference
        residual = self.relu(self.conv1(x))
        residual = self.relu(self.conv2(residual))
        residual = self.conv3(residual)
        return x + residual    # LR + residual = SR
```

> Residual learning is the key trick in super-resolution: the network doesn't directly predict the HR image, but learns the difference between HR and LR. This dramatically reduces learning difficulty since residual images are near-zero in most regions.

### 3.3 Loss Functions

| Loss | Formula | Characteristics |
|------|---------|-----------------|
| MSE | $\frac{1}{N}\sum(y - \hat{y})^2$ | Optimizes PSNR, may over-smooth |
| L1 | $\frac{1}{N}\sum\|y - \hat{y}\|$ | More robust to outliers |
| Perceptual | Based on VGG feature distance | Better perceptual quality, lower PSNR |

This experiment uses **MSE Loss**, simple and direct.

## IV. Data Preparation

### 4.1 Constructing LR-HR Training Pairs

```python
from PIL import Image
import torchvision.transforms as transforms

def create_lr_hr_pair(hr_image, scale=2):
    """Generate LR-HR training pair from HR image"""
    h, w = hr_image.shape[1:]  # (C, H, W)

    # 1. Downsample to get true LR
    lr = transforms.functional.resize(
        hr_image, [h // scale, w // scale],
        interpolation=transforms.InterpolationMode.BICUBIC
    )

    # 2. Upsample back to original size as network input
    lr_up = transforms.functional.resize(
        lr, [h, w],
        interpolation=transforms.InterpolationMode.BICUBIC
    )

    return lr_up, hr_image  # (input, target)
```

### 5.1 Training Configuration

```python
SCALE = 2                 # Upscaling factor

EPOCHS = 10

```

### 5.2 Training Loop

```python
def train(model, dataloader, criterion, optimizer, epochs, scale):
    model.train()
    losses = []

    for epoch in range(epochs):
        running_loss = 0.0

        for images, _ in dataloader:
            # Construct LR-HR pairs
            lr_up, hr = create_lr_hr_batch(images, scale)

            # Forward pass
            sr = model(lr_up)
            loss = criterion(sr, hr)

            # Backward pass
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

            running_loss += loss.item()
            losses.append(loss.item())

        avg_loss = running_loss / len(dataloader)
        print(f"Epoch [{epoch+1}/{epochs}] Loss: {avg_loss:.6f}")

    return losses
```

### 5.3 Training Key Points

| Point | Explanation |
|-------|-------------|
| Residual learning | Network output added to LR input, learns residual not full image |
| Data construction | Dynamically create LR-HR pairs each iteration, no pre-storage needed |
| PSNR monitoring | Compute PSNR during training, observe reconstruction quality improvement |
| GPU acceleration | CIFAR-10 is small, trainable on CPU, faster on GPU |

## VI. Evaluation and Visualization

### 6.1 PSNR Computation

```python
def compute_psnr(sr, hr):
    """Compute PSNR"""
    mse = torch.mean((sr - hr) ** 2)
    if mse == 0:
        return float('inf')
    return 10 * torch.log10(1.0 / mse).item()
```

### 6.2 Result Comparison

```python
def visualize_results(lr_images, sr_images, hr_images, num_samples=4):
    """Side-by-side comparison: LR → SR → HR"""
    fig, axes = plt.subplots(num_samples, 3, figsize=(12, 4*num_samples))

    titles = ['Low Resolution (Input)', 'Super Resolution (Output)', 'High Resolution (Ground Truth)']

    for i in range(num_samples):
        axes[i, 0].imshow(lr_images[i].permute(1,2,0).clip(0,1))
        axes[i, 1].imshow(sr_images[i].permute(1,2,0).detach().clip(0,1))
        axes[i, 2].imshow(hr_images[i].permute(1,2,0).clip(0,1))

        for j in range(3):
            axes[i, j].set_title(titles[j] if i == 0 else '')
            axes[i, j].axis('off')

    plt.tight_layout()
    plt.savefig('sample_results.png', dpi=150)
    plt.show()
```
