## 二、什么是图像超分辨率

### 2.1 问题定义

图像超分辨率的目标是：**从低分辨率图像重建出高分辨率图像**。

$$I_{HR} \approx f_\theta(I_{LR})$$

其中 $I_{LR}$ 是低分辨率输入，$f_\theta$ 是神经网络，$I_{HR}$ 是高分辨率目标。

### 2.2 退化模型

低分辨率图像通常由高分辨率图像经过退化过程得到：

$$I_{LR} = D(I_{HR}) = (I_{HR} * k) \downarrow_s + n$$

| 因素 | 符号 | 说明 |
|------|------|------|
| 模糊核 | $k$ | 光学系统或运动导致的模糊 |
| 下采样 | $\downarrow_s$ | 以倍数 $s$ 降低分辨率 |
| 噪声 | $n$ | 传感器噪声 |

在本次实验中，我们简化为**双三次下采样 + 双三次上采样**来构造 LR-HR 训练对。

### 2.3 评价指标

| 指标 | 全称 | 含义 | 范围 |
|------|------|------|------|
| PSNR | Peak Signal-to-Noise Ratio | 像素级重建质量 | 越高越好，通常 20-40 dB |
| SSIM | Structural Similarity | 结构相似度 | 0~1，越接近 1 越好 |

$$PSNR = 10 \cdot \log_{10}\left(\frac{MAX_I^2}{MSE}\right)$$

## 三、方法原理

### 3.1 插值法（传统基线）

```python
# 双三次插值 —— 传统方法，作为基线对比
lr_image = cv2.imread('lr.png')
sr_bicubic = cv2.resize(lr_image, None, fx=2, fy=2, interpolation=cv2.INTER_CUBIC)
```

| 方法 | 优点 | 缺点 |
|------|------|------|
| 最近邻 | 速度快 | 锯齿严重 |
| 双线性 | 平滑 | 边缘模糊 |
| 双三次 | 较好平滑 | 高频细节丢失 |

### 3.2 基于 CNN 的超分辨率

卷积神经网络能从数据中学习 LR→HR 的映射，恢复插值法无法重建的高频细节：

```python
class SRCNN(nn.Module):
    """经典超分辨率卷积神经网络"""
    def __init__(self):
        super().__init__()
        # 特征提取
        self.conv1 = nn.Conv2d(3, 64, kernel_size=5, padding=2)
        # 非线性映射
        self.conv2 = nn.Conv2d(64, 64, kernel_size=3, padding=1)
        # 重建
        self.conv3 = nn.Conv2d(64, 3, kernel_size=5, padding=2)
        self.relu = nn.ReLU()

    def forward(self, x):
        # 残差学习：学习 HR - LR 的差值
        residual = self.relu(self.conv1(x))
        residual = self.relu(self.conv2(residual))
        residual = self.conv3(residual)
        return x + residual    # LR + 残差 = SR
```

> 残差学习是超分辨率的关键技巧：网络不直接预测 HR 图像，而是学习 HR 与 LR 之间的差异。这大幅降低了学习难度，因为残差图像大部分区域接近零。

### 3.3 损失函数

| 损失函数 | 公式 | 特点 |
|---------|------|------|
| MSE | $\frac{1}{N}\sum(y - \hat{y})^2$ | 优化 PSNR，可能过度平滑 |
| L1 | $\frac{1}{N}\sum|y - \hat{y}|$ | 对异常值更鲁棒 |
| Perceptual | 基于 VGG 特征距离 | 感知质量更好，但 PSNR 可能更低 |

本次实验使用 **MSE Loss**，简单直接。

## 四、数据准备

### 4.1 构造 LR-HR 训练对

```python
from PIL import Image
import torchvision.transforms as transforms

def create_lr_hr_pair(hr_image, scale=2):
    """从 HR 图像生成 LR-HR 训练对"""
    h, w = hr_image.shape[1:]  # (C, H, W)

    # 1. 下采样得到真正的 LR
    lr = transforms.functional.resize(
        hr_image, [h // scale, w // scale],
        interpolation=transforms.InterpolationMode.BICUBIC
    )

    # 2. 上采样回原尺寸作为网络输入
    lr_up = transforms.functional.resize(
        lr, [h, w],
        interpolation=transforms.InterpolationMode.BICUBIC
    )

    return lr_up, hr_image  # (输入, 目标)
```

## 五、模型训练

### 5.1 训练配置

```python
# 超参数
SCALE = 2                 # 放大倍数
BATCH_SIZE = 32
LEARNING_RATE = 0.001
EPOCHS = 10

model = SRCNN()
criterion = nn.MSELoss()
optimizer = torch.optim.Adam(model.parameters(), lr=LEARNING_RATE)
```

### 5.2 训练循环

```python
def train(model, dataloader, criterion, optimizer, epochs, scale):
    model.train()
    losses = []

    for epoch in range(epochs):
        running_loss = 0.0

        for images, _ in dataloader:
            # 构造 LR-HR 对
            lr_up, hr = create_lr_hr_batch(images, scale)

            # 前向传播
            sr = model(lr_up)
            loss = criterion(sr, hr)

            # 反向传播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

            running_loss += loss.item()
            losses.append(loss.item())

        avg_loss = running_loss / len(dataloader)
        print(f"Epoch [{epoch+1}/{epochs}] Loss: {avg_loss:.6f}")

    return losses
```

### 5.3 训练要点

| 要点 | 说明 |
|------|------|
| 残差学习 | 网络输出加到 LR 输入上，学习残差而非完整图像 |
| 数据构造 | 每次迭代动态创建 LR-HR 对，无需预存 |
| PSNR 监控 | 训练同时计算 PSNR，观察重建质量提升 |
| GPU 加速 | CIFAR-10 规模小，CPU 也可训练，GPU 更快 |

## 六、评估与可视化

### 6.1 PSNR 计算

```python
def compute_psnr(sr, hr):
    """计算 PSNR"""
    mse = torch.mean((sr - hr) ** 2)
    if mse == 0:
        return float('inf')
    return 10 * torch.log10(1.0 / mse).item()
```

### 6.2 结果对比

```python
def visualize_results(lr_images, sr_images, hr_images, num_samples=4):
    """对比显示 LR → SR → HR"""
    fig, axes = plt.subplots(num_samples, 3, figsize=(12, 4*num_samples))

    titles = ['Low Resolution (Input)', 'Super Resolution (Output)', 'High Resolution (Ground Truth)']

    for i in range(num_samples):
        axes[i, 0].imshow(lr_images[i].permute(1,2,0).clip(0,1))
        axes[i, 1].imshow(sr_images[i].permute(1,2,0).detach().clip(0,1))
        axes[i, 2].imshow(hr_images[i].permute(1,2,0).clip(0,1))

        for j in range(3):
            axes[i, j].set_title(titles[j] if i == 0 else '')
            axes[i, j].axis('off')

    plt.tight_layout()
```
