## 一、具身智能中的传感器

具身智能体通过传感器感知物理世界。常见传感器类型：

| 传感器     | 感知维度         | 典型应用           |
| ---------- | ---------------- | ------------------ |
| RGB 摄像头 | 2D 彩色图像      | 目标检测、语义分割 |
| 深度摄像头 | 3D 点云 / 深度图 | 避障、抓取定位     |
| 红外摄像头 | 热辐射           | 暗光/夜间感知      |
| 激光雷达   | 3D 高精度点云    | 建图、定位         |

## 二、视觉任务分层

视觉任务按抽象层次分为两类：

- **底层视觉任务**：像素级处理，如边缘检测、光流、超分辨率
- **高层视觉任务**：语义级理解，如目标识别、场景理解、行为分析

## 三、从问题到方法的演进

核心问题：**How to find the function?**

面对一个问题，方法选择的逻辑：

```
问题 → 你知道怎么做吗？
        ├─ 知道 → 专家系统（规则驱动）
        └─ 不知道 → 容易建模？
                      ├─ 是 → 机器学习
                      └─ 否 → 强化学习
```

## 四、机器学习方法谱系

从手动特征到自动学习的演进：

```
特征工程 → 浅层学习 → 表示学习 → 深度学习
(人工设计)   (SVM等)    (自动提取)   (端到端)
```

- **特征工程**：人工设计特征（如 HOG、SIFT），依赖领域知识
- **浅层学习**：SVM、决策树等，特征仍需手动设计
- **表示学习**：让模型自动学习特征表示
- **深度学习**：多层网络端到端学习，从原始输入直接到输出

## 五、神经网络基础

- **赫布法则**（Hebb's Rule）："一起激活的神经元连接会增强"——神经网络学习的生物学基础
- **人工神经元**：模拟生物神经元的数学模型（加权求和 + 激活函数）
- **人工神经网络**：多层人工神经元的组合，构成深度学习的基础

## 六、课程观点

> 具身智能不是风口，而是趋势。
