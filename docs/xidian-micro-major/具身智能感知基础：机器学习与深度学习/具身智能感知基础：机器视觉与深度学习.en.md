## 1. Sensors in Embodied Intelligence

An embodied agent perceives the physical world through sensors. Common sensor types:

| Sensor | Perception Dimension | Typical Application |
|--------|---------------------|---------------------|
| RGB Camera | 2D color images | Object detection, semantic segmentation |
| Depth Camera | 3D point clouds / depth maps | Obstacle avoidance, grasp localization |
| Infrared Camera | Thermal radiation | Low-light / nighttime perception |
| LiDAR | High-precision 3D point clouds | Mapping, localization |

## 2. Hierarchy of Vision Tasks

Vision tasks are divided into two levels of abstraction:

- **Low-level vision tasks**: Pixel-level processing — edge detection, optical flow, super-resolution.
- **High-level vision tasks**: Semantic-level understanding — object recognition, scene understanding, behavior analysis.

## 3. From Problem to Method: An Evolution

The core question: **How to find the function?**

When facing a problem, the logic for method selection:

```
Problem → Do you know how to solve it?
            ├─ Yes → Expert System (rule-driven)
            └─ No  → Easy to model?
                       ├─ Yes → Machine Learning
                       └─ No  → Reinforcement Learning
```

## 4. The Machine Learning Spectrum

The evolution from hand-crafted features to automatic learning:

```
Feature Engineering → Shallow Learning → Representation Learning → Deep Learning
  (hand-designed)       (SVM, etc.)         (automatic extraction)    (end-to-end)
```

- **Feature Engineering**: Hand-designed features (e.g., HOG, SIFT) — depends on domain expertise.
- **Shallow Learning**: SVM, decision trees, etc. — features still need manual design.
- **Representation Learning**: Let the model automatically learn feature representations.
- **Deep Learning**: Multi-layer networks learn end-to-end, from raw input directly to output.

## 5. Neural Network Basics

- **Hebb's Rule**: "Neurons that fire together wire together" — the biological foundation of neural network learning.
- **Artificial Neuron**: A mathematical model simulating biological neurons (weighted sum + activation function).
- **Artificial Neural Network**: A composition of multiple layers of artificial neurons — the foundation of deep learning.
