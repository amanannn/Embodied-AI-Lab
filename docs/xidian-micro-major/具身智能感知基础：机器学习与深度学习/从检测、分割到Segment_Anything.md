## 一、视觉任务的粒度递进

| 任务 | 粒度 | 回答 | 输出 |
|------|------|------|------|
| 分类 | 图像级 | "图里有什么？" | 类别标签 |
| 检测 | 区域级 | "物体在哪？" | 边界框 |
| 语义分割 | 像素级 | "每个像素属于哪类？" | 逐像素类别 |
| 实例分割 | 像素+ID | "每个物体实例的边界？" | 逐像素 Mask + 实例 ID |
| **通用分割** | **任意粒度** | **"你指哪，我分哪"** | **Prompt → Mask** |

## 二、语义分割

### 2.1 定义

将图像中每个像素分配一个语义类别——"天空区域的像素标蓝，道路区域的像素标灰"。

### 2.2 经典方法

| 方法 | 年份 | 核心思路 |
|------|------|---------|
| **FCN** | 2015 | 全卷积网络——用卷积替代全连接，输出像素级预测 |
| **U-Net** | 2015 | 编码器-解码器 + 跳跃连接——医疗图像标配 |
| **DeepLab** | 2016-2018 | 空洞卷积 + ASPP + CRF 后处理 |
| **PSPNet** | 2017 | 金字塔池化捕获多尺度上下文 |

> FCN 的历史意义：第一次证明"全卷积"可以端到端做像素级预测。

### 2.3 语义分割的局限

| 局限 | 说明 |
|------|------|
| **不分实例** | 并排的两只猫 → 同一类别标签，无法区分个体 |
| **固定类别** | 只能分割训练集定义好的类别 |
| **无交互** | 无法按用户意图指定分割目标 |

---

## 三、实例分割

### 3.1 定义

在语义分割的基础上区分每个实例——"这是第一只猫，这是第二只猫"。

### 3.2 代表方法

| 方法 | 年份 | 思路 |
|------|------|------|
| **Mask R-CNN** | 2017 | Faster R-CNN + 分割分支——检测 + 分割二合一 |
| **YOLACT** | 2019 | 实时实例分割（30+ FPS） |
| **SOLO** | 2020 | 按位置区分实例——无需检测框 |

> Mask R-CNN 三输出：类别 + 框 + Mask。三头共享骨干网络，是检测到分割的范式桥梁。

## 四、SAM——通用分割

### 4.1 SAM 之前的困境

| 问题 | 传统方案 |
|------|---------|
| 要分割一个新类别 | 标注几千张图，训练专用模型 |
| 要分割"左边那个红色的" | 无法用类别名描述 |
| 要交互式分割 | 需要专门训练的交互工具 |

### 4.2 SAM 的核心设计

**Promptable Segmentation**——"你给提示，我出 Mask"：

| 提示类型 | 示例 | SAM 如何处理 |
|---------|------|------------|
| **点** | 点击物体的一个位置 | 从该点向外"生长"Mask |
| **框** | 画一个矩形 | 框内区域做分割 |
| **文本** | "the red cup" | 配合 CLIP/Grounding DINO |
| **Mask** | 粗糙涂抹 | 精修为精确边界 |

### 4.3 SAM 架构

```
输入图像
    ↓
Image Encoder（ViT） → 图像嵌入（一次，可复用）
    ↓
Prompt Encoder → 提示嵌入（点/框/文本/Mask）
    ↓
Mask Decoder → 输出 Mask + 置信度分数
```

| 组件 | 作用 | 特点 |
|------|------|------|
| **Image Encoder** | ViT 提取图像特征 | 最重的部分，只需跑一次，可缓存 |
| **Prompt Encoder** | 编码用户提示 | 轻量，实时响应 |
| **Mask Decoder** | 融合图像+提示→Mask | 可输出多个候选 Mask |

### 4.4 SAM 为什么重要？

| 意义 | 说明 |
|------|------|
| **数据飞轮** | 用 SAM 自动标注 → 训练专用模型 → 更多数据 |
| **零样本分割** | 不需要训练就能分割任意物体 |
| **可组合** | SAM + CLIP = 开放词汇分割；SAM + Grounding DINO = 文本→Mask |
| **具身智能基石** | 机器人抓取前需要精确的目标 Mask |

> SAM 是视觉基础模型的分水岭——首次证明了**通用分割**的可行性。它不是"又一个分割模型"，而是分割从"任务"变成了"能力"。

## 五、从检测到 SAM 的完整链路

```
图像
  ↓
【检测】R-CNN / YOLO / DETR → 找到物体框
  ↓
【分类】ResNet / ViT → 判断框内是什么
  ↓
【分割】FCN / Mask R-CNN → 逐像素标注
  ↓
【通用分割】SAM → 任意提示 → 任意物体 Mask
  ↓
【开放词汇】CLIP + SAM → 自然语言 → 指定物体的 Mask
  ↓
【具身执行】Mask → 3D 投影 → 抓取规划
```
