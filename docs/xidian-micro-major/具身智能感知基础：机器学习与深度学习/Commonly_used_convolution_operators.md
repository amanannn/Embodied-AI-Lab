什么是卷积算子？ 特征提取和表示学习

如何实现卷积算子？

传统3乘3卷积：功能单一，参数量偏大，感受野较小

卷积变体：实现特征维度变换，降低参数量，增加卷积感受野


常见的卷积算子

1.1乘1卷积：与传统的3乘3卷积操作相同，采用感受野为1乘1的卷积核来运算

```Python
conv = nn.Conv2d(in_channels=512, out_channels=256, kernel_size=1)
```

2.池化层：改变输入特征的尺寸，主要用于图像的降采样，常见的池化方法有平均池化，最大值池化，自适应池化等

```Python
pool = nn.AvgPool2d(kernel_size=2, stride=2)
```

实现特征降维 保持特征不变性 减少过拟合（补充：深度学习的3个状态：拟合，欠拟合，过拟合）

3.转置卷积：反卷积，目的是上采样，由较小尺寸的输入得到大分辨率输出。其运算过程可以等价于先对输入进行padding再做卷积

```Python
conv = nn.ConvTranspose2d(in_channels, out_channels, kernel_size=3, stride=1, padding=0)
```

4.分组卷积：输入和输出的通道会被划分为多个组，每个组的输出通道只和对应组内的输入通道相连接。而与其他组的通道无关。仅需要不分组参数量的1/g（g为分组数）

```Python
conv = nn.Conv2d(in_channels=24, out_channels=48, kernel_size=3, groups=3)
```

常规3乘3卷积参数量 = 3乘3乘24乘18 =10368

分组卷积参数量 = 3乘（3乘3乘8乘18） = 10368/3 = 3456

5,深度可分离卷积：由每个通道为一组的分组卷积与1乘1卷积组成，其中1乘其卷积负责将深度卷积的输出通道投影到一个新的特征图上

```Python
depthwise_conv = nn.Conv2d(in_channels=24, outchannels=24, kernel_size=3, groups=24)
pointwise_conv = nn.Conv2d(in_channels=24, outchannels=48, kernel_size=1)
```

降低参数量 实现信息的跨通道交互与整合

6.空洞卷积：可以看成是大卷积更进一步的扩展，在原始卷积的基础上引入空洞率这个超参数，用于调节卷积核的间隔距离

```Python
conv = nn.Conv2d(in_channels, out_channels, kernel_size=3, dilation=2)
```

增大感受野：在不增加参数的基础上，获得更大的感受野 多尺度分析

7.可变形卷积：对卷积进行局部形变，从而适应野外区域的不规则形状。在感受野中引入了偏移量进行学习来修正感受野

输入张量 -> 通过卷积计算偏置（或者跳过这一步） -> 双线性质插值 -> 卷积 -> 输入张量

双线性差值：当你要获取的像素位置不在整齐的网格点上（例如坐标是 (3.7, 5.2)），而是落在四个整数像素之间时，双线性插值通过计算该点周围 4个最近邻整数像素 的加权平均值，来“合成”出这个位置的数值。（对线性差值在二维直角网格上的扩展，在两个方向分别进行线性插值）

适应不规则结构 减少网格偏差
