# 什么是卷积网络（CNN）

> 西安电子科技大学 · 具身智能微专业
> 课程笔记

## 一、从一维卷积开始

### 1.1 一维离散卷积

卷积运算是信号处理的核心操作。一维离散卷积定义：

$$y[t] = \sum_{k=-\infty}^{\infty} x[k] \cdot h[t-k]$$

直观理解：**卷积核 $h$ 在输入信号 $x$ 上滑动，每个位置做逐元素相乘再求和**。

### 1.2 卷积运算的意义

> 卷积可以提取信号中某种特性的信息（特征）。

不同的卷积核提取不同的特征——边缘、纹理、角点，或更高层的语义模式。

## 二、二维卷积

图像是二维信号，卷积核也是二维的：

$$\text{output}(i,j) = \sum_{m}\sum_{n} \text{input}(i+m, j+n) \cdot \text{kernel}(m, n)$$

| 超参数 | 含义 |
|--------|------|
| 卷积核大小（Kernel Size） | 每次运算涉及的邻域范围，如 3×3、5×5 |
| 步长（Stride） | 卷积核每次移动的像素数 |

从**人为设计卷积核**（Sobel、Laplacian）到**参数可学习的卷积核**——这是传统图像处理到深度学习的核心跨越。

## 三、为什么图像处理要用卷积而非全连接？

如果直接把图像拉成一维向量送入全连接网络：

| 全连接网络的问题 | 卷积网络如何解决 |
|-----------------|----------------|
| 参数量爆炸（224×224×3 ≈ 150k 输入神经元） | 卷积核共享参数，参数量与输入尺寸无关 |
| 空间信息丢失（像素之间的邻域关系被抹平） | 卷积天然保留 2D 空间结构 |
| 无平移不变性 | 卷积核滑过整张图，对平移不敏感 |

卷积网络的核心优势：

- **稀疏连接**：每个神经元只连接局部区域（局部特征），深层组合成全局特征
- **参数共享**：同一卷积核在整个图像上复用
- **平移不变性**：无论特征出现在图像的哪个位置，都能被检测到

## 四、经典 CNN 架构演进

### 4.1 LeNet-5（1998）

手写数字识别的开山之作。结构：卷积 → 池化 → 卷积 → 池化 → 全连接 → 输出。

### 4.2 AlexNet（2012）

ImageNet 竞赛冠军，首次将深度 CNN 推向主流。关键改进：ReLU 激活、Dropout、GPU 训练。

### 4.3 VGG（2014）

核心思想：**用多个小卷积核（3×3）堆叠替代大卷积核**。

> 两个 3×3 卷积堆叠 = 一个 5×5 卷积的感受野，但参数量更少，非线性更强。

### 4.4 GoogLeNet / Inception（2014）

引入 **Inception 结构**：在同一层并行使用不同尺寸的卷积核（1×1、3×3、5×5）和池化，融合多尺度特征信息。

```
输入 → ┬─ 1×1 卷积 ───→
       ├─ 3×3 卷积 ───→  拼接 → 输出
       ├─ 5×5 卷积 ───→
       └─ 3×3 最大池化 →
```

### 4.5 ResNet（2015）

**核心问题**：网络加深时，梯度消失/爆炸导致训练困难。

| 问题 | 表现 |
|------|------|
| 梯度消失 | 浅层梯度趋近于 0，参数几乎不更新 |
| 梯度爆炸 | 浅层梯度极大，参数更新震荡 |

**残差结构（Residual Block）**：

$$y = \mathcal{F}(x) + x$$

> 学习 100 → 101 很难，但学习 0 → 1 很简单。残差连接让网络只需要学习"增量"。

效果：解决了深层网络的训练问题，加速收敛，实现浅层特征复用。

## 六、CNN 的计算实例

以一个简单网络为例：

```
输入: 3×3×1 (高度×宽度×通道)
  ↓ 卷积层: 64 个 3×3×1 卷积核 → W[64, 1, 3, 3], B[64]
输出特征图: 3×3×64
```

更一般地（需要 padding 保持尺寸）：

```
输入 [1, 5, 5]（通道=1, H=5, W=5）
  ↓ Conv1: in=1, out=16, kernel=3, padding=1 → W[16, 1, 3, 3], B[16]
输出 [16, 5, 5]
  ↓ Conv2: in=16, out=N, kernel=3, padding=1 → W[N, 16, 3, 3], B[N]
输出 [N, 5, 5]
```

> 特征图尺寸约定：[通道数, 高度, 宽度]

**关键结论**：处理大尺寸二维矩阵时，小卷积核的卷积网络远优于全连接——参数量可控、空间结构保留、特征层次化。
