# MAE——掩码自编码器

> 西安电子科技大学 · 具身智能微专业
> 课程笔记
> 论文：Masked Autoencoders Are Scalable Vision Learners (He et al., CVPR 2022)

---

## 一、动机：视觉预训练为什么落后于 NLP？

| 维度 | NLP（BERT） | CV（MAE 之前） |
|------|------------|---------------|
| **自监督任务** | MLM——遮词预测，天然自监督 | 依赖 ImageNet 标签或对比学习 |
| **数据效率** | 极高——每个 batch 有 15% token 被预测 | 相对低 |
| **架构统一** | Transformer 天然适合 | CNN 主导，ViT 刚出现 |
| **规模化** | GPT-3 证明了 Scaling Law | 视觉模型规模远落后 |

> MAE 的核心洞察：视觉信号的冗余度远高于文本——遮掉 75% 的图像 patch，人类仍能理解内容。这意味着**极高掩码率的自监督重建**是可行的。

---

## 二、MAE 核心设计

### 2.1 非对称编解码器

| 组件 | 设计 | 计算量 |
|------|------|--------|
| **Encoder** | 标准 ViT，只处理**可见 patch**（25%） | 重——但只处理 1/4 的 patch |
| **Decoder** | 轻量 Transformer（层数少、宽度窄），处理**全部 patch**（可见 + 掩码） | 轻——预训练结束后直接丢弃 |

### 2.2 75% 极高掩码率

| 掩码率 | 含义 | MAE 采用 |
|--------|------|---------|
| 15%（BERT） | 每 7 个词遮 1 个 | 不适合图像——周围像素可轻易推断被遮内容 |
| 50% | 一半可见 | 仍然太容易 |
| **75%（MAE）** | **3/4 被遮，仅 1/4 可见** | ✅ 强制模型学习全局语义 |

> BERT 只需要 15% 掩码率，因为文本语义密集；图像像素冗余极高，遮 75% 仍然可重建。这是 MAE 和 BERT 最本质的区别。

### 2.3 训练流程

```
输入图像 (224×224)
    ↓ Patchify —— 切为 196 个 patch (16×16)
    ↓ Random Mask —— 保留 25%（49 个）可见 patch
    ↓
Encoder（ViT）—— 只编码可见 patch → 得到可见 patch 的特征
    ↓
Decoder —— 可见 patch 特征 + 掩码 token → 重建全部 196 个 patch
    ↓
Loss —— 仅对掩码 patch 计算 MSE（不惩罚可见 patch 的重建）
```

---

## 三、关键技术细节

### 3.1 为什么 Encoder 只处理可见 patch？

| 方案 | 计算量 | 效果 |
|------|--------|------|
| Encoder 处理全部 patch（可见+掩码 token） | $O(N^2)$，N=196 | 浪费——大量计算花在掩码 token 上 |
| **Encoder 只处理可见 patch** | $O((N/4)^2)$ | 减少 16× 计算量 |

> 这是 MAE 高效的关键——Encoder 只跑 25% 的 token，Decoder 才是轻量的全量重建。

### 3.2 损失函数

$$L = \frac{1}{N_{mask}}\sum_{i \in masked} \| \hat{p}_i - p_i \|^2$$

- 只对被掩码 patch 计算 MSE
- $p_i$ 是原始像素值（归一化后），$\hat{p}_i$ 是重建像素值
- 也可以用特征空间的损失（如 CLIP 特征），效果更好但更复杂

### 3.3 可视化重建效果

训练好的 MAE 可以用 25% 的可见 patch 重建出合理的完整图像——证明模型学到了全局语义理解。

## 四、MAE 与 BERT 的核心差异

| 维度 | BERT | MAE |
|------|------|-----|
| **模态** | 自然语言 | 图像 |
| **信息密度** | 高（每个词承载丰富语义） | 低（相邻像素几乎相同） |
| **掩码率** | 15% | 75% |
| **Encoder 处理** | 全部 token（包括 [MASK]） | 仅可见 patch |
| **Decoder** | 无（MLM 是分类任务，直接输出） | 有（像素重建是回归任务） |
| **预训练任务** | 词级分类 | 像素级回归 |

---

## 五、MAE 的影响

| 领域 | 影响 |
|------|------|
| **自监督学习** | 证明了高掩码率 + 非对称架构的可行性 |
| **ViT 普及** | MAE 是 ViT 的最佳预训练搭档 |
| **多模态** | MAE 思想可扩展到视频（VideoMAE）、点云 |
| **计算效率** | 仅 1/4 Token 进 Encoder → 大幅降低预训练成本 |

---

## 六、局限与后续工作

| 局限 | 后续探索 |
|------|---------|
| 重建的像素损失不直接等用于下游任务性能 | 特征空间重建（如 CLIP 特征替代像素） |
| 固定掩码率 | 自适应掩码策略 |
| 仅做图像 | VideoMAE、MultiMAE（多模态） |
