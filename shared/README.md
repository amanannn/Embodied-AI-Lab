# Shared Modules

English: [README.en.md](./README.en.md)

## 作用

这个目录用于承接跨方向、跨实验真正可复用的基础设施。它的目的不是“把文件挪得更整齐”，而是把已经稳定、已经有复用价值的通用能力从单个实验里抽离出来。

## 纳入规则

### 什么应该进入 `shared/`

只有在下面条件成立时，代码或文档才应该进入 `shared/`：

- 至少会被两个方向复用
- 接口已经稳定到可以作为依赖
- 不需要依赖某个单独实验的教学叙事才能理解

### 什么应该留在方向实验里

以下内容不应为了“看起来整洁”而提前抽到 `shared/`：

- 单个实验专属脚本
- 仍在快速变化的私有接口
- 只在一个方向上下文里有意义的结论、奖励函数、调参逻辑或实验写法

## 稳定性与复用预期

进入 `shared/` 的内容应被视为跨方向契约。它的命名、文件布局和行为稳定性，应高于实验内部辅助代码。

## 当前模块

- `sim2d/`：导航、控制与 sim-to-real 方向共享的轻量仿真基础
- `viz/`：跨实验复用的绘图、动画与展示工具
- `math/`：稳定的状态估计、几何与工具数学模块
- `ros2_interfaces/`：混合语言实验使用的 ROS2 边界与接口
- `datasets/`：跨方向复用的数据、元数据和下载说明
