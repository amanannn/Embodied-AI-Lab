# datasets

中文： [README.md](./README.md)

This module will collect lightweight packaged datasets or dataset notes that multiple direction labs depend on.

## Intended Ownership

- small bundled data files
- download instructions for larger public datasets
- metadata notes for cross-direction reuse

## Out of Scope

The following does not belong here:

- large raw dumps that are only used by one experiment
- private annotation workflows for a single research branch
- transient scratch outputs, checkpoints, or evaluation artifacts
